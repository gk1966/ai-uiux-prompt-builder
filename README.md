AI UI/UX Prompt Builder – Educational Edition v3.4
A web-based educational tool that scaffolds structured, theory-grounded prompt construction for AI-generated UI/UX components.
Overview
The AI UI/UX Prompt Builder transforms abstract design knowledge into actionable, structured prompts for generative AI systems. Rather than treating prompt writing as freeform experimentation, the tool guides users through a five-stage workflow grounded in established HCI theory — producing pedagogically annotated prompts that embed design rationale directly into the AI instruction.
Key Features
Five-Stage Prompt Construction: Component Selection → Aesthetic Definition → Theoretical Alignment (Nielsen, Norman, Gestalt) → Accessibility Specification (WCAG 2.1/2.2) → Pedagogical Annotation
Integrated WCAG Contrast Checker: Real-time color contrast verification against accessibility standards
Quality Scoring Algorithm: Real-time evaluation of prompt comprehensiveness across multiple dimensions
Dual Output Workflow: Universal copyable prompt for any AI system + optional direct Gemini API integration for immediate SVG generation
SVG Output: Scalable, editable vector graphics compatible with Adobe Illustrator, Figma, Sketch, and all modern browsers
Comparison Mode: Side-by-side display of naive vs. theory-grounded prompts for reflective learning
Ready-Made Templates: Quick-start templates for common UI components
Theoretical Foundations
Nielsen's 10 Usability Heuristics (Nielsen, 1994)
Norman's Design Principles: visibility, feedback, affordance (Norman, 2013)
Gestalt Perceptual Principles (Wertheimer, 1923)
WCAG 2.1/2.2 Accessibility Standards (W3C)

Usage
Open `index.html` in any modern web browser. No installation, no dependencies, no server required — the application runs entirely client-side.
Educational Context
The tool supports prompt literacy as a core design competency. It is accompanied by a structured laboratory protocol (From Abstract Theory to Structured Prompts) for classroom integration, guiding students from naive prompting to theory-grounded prompt construction through scaffolded exercises.
