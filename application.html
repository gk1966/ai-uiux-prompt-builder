UI/UX Design Principles 📚 Nielsen's Heuristics 📚 Norman's Principles

📖 About Nielsen's Usability Heuristics

Jakob Nielsen's 10 usability heuristics are broad rules of thumb for user interface design.

Reference: Nielsen, J. (1994). "Enhancing the explanatory power of usability heuristics."

📖 About Norman's Design Principles

Don Norman's principles emphasize affordance, feedback, and visibility.

Reference: Norman, D. A. (2013). "The Design of Everyday Things."

Similar elements should look and behave the same way.
Users should immediately understand what they can do.
Every action should produce a visible response.
Show options instead of making users remember them.
Design to prevent user errors before they happen.
Enable both novice and expert users to work quickly.
Guide attention to the most important elements first.
Buttons should look pressable, inputs should look editable.

Gestalt Perceptual Principles 📚 Gestalt Psychology

📖 About Gestalt Principles

They describe how humans naturally perceive and organize visual elements.

Reference: Wertheimer, M. (1923). "Laws of Organization in Perceptual Forms."

Group related elements by placing them close together.
Use consistent styling for elements with similar functions.
Simplified shapes that users mentally complete.
Create clear distinction between content and background.
Arrange elements to guide the user's eye naturally.
Synchronized movement indicates grouping.

Accessibility Standards WCAG 2.1/2.2

Minimum 4.5:1 for normal text.
Visible focus states for keyboard navigation.
Minimum 44x44px for touch targets.
Semantic HTML and ARIA attributes for screen readers.
Don't rely on color alone to convey information.
Minimum 16px for body text, scalable typography.

Technical Parameters & Export Options

e.g. 3 button types: Primary, Secondary, Ghost
